# temporal-network-automation

## Getting Started
- Follow the Quickstart guide here to run your first workflow: https://docs.temporal.io/develop/python/set-up-your-local-python
- [Click here to download the Temporal zip file](temporal.zip)
- Once downloaded, start your temporal server via command prompt or CLI
  - `temporal server start-dev`
- The default values for your server:
  - UI: http://localhost:8233/namespaces/default/workflows
  - gRPC: localhost:7233 (where your client will connect to in your worker config)
  - Metrics: http://localhost:61170/metrics
